### Syntax
 
`set(obj, name, value)`
 
`obj.set(name, value)`
 
### Description
 
`set(obj, name, value)` sets the preference `name` to the
value given by `value`
 
`obj.set(name, value)` sets the preference `name` to the
value given by `value`
